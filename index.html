The model can be exported for applications such as mobile apps for real-time flower recognition. Demonstrated skills in data wrangling, deep learning, computer vision, and AI model deployment.
  • Data Science & Data Wrangling Projects (Udacity Nanodegree): Python-based projects focusing on cleaning, transforming, and analyzing complex datasets. Conducted exploratory data analysis, handled missing or inconsistent data, and applied data wrangling techniques to prepare datasets for visualization and modeling. Skills demonstrated include Pandas, NumPy, Matplotlib, and generating actionable insights from raw data.
  • AI-Driven Automated Messaging Platform: Python web platform for scheduling AI-generated messages delivered automatically to personal phones via Gmail. Demonstrated skills in Python, web automation, and AI integration.
  • AI Storytelling Using ChatGPT (Purdue): Developed an interactive storytelling platform leveraging ChatGPT to create unique, engaging narratives without coding. Users collaboratively build stories, while AI generates responses that expand the plot and guide decision-making. Key objectives and workflow:
    • Problem Scenario: Create an interactive storytelling adventure using ChatGPT to generate narratives collaboratively and creatively.
    • Objective: Unleash creativity and storytelling abilities through AI-assisted narrative generation.
    • Steps Performed:
      • Define the story theme (e.g., fantasy, sci-fi, mystery, adventure).
      • Introduce main characters with names, personalities, and backgrounds.
      • Set the scene to kickstart the story.
      • Accept participant input to influence character actions and plot direction.
      • Generate AI responses with ChatGPT to advance the story.
      • Implement decision-making and branching paths, allowing multiple outcomes.
      • Iterate and refine user inputs for clarity and consistency.
      • Guide the story to a satisfying conclusion or an open-ended continuation.
      • Showcase and share the interactive narrative.
